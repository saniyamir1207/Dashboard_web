# Dashboard_web

# 📊 Dataset Intelligence Dashboard

A modern **Streamlit-based Data Analytics Dashboard** that enables users to upload any CSV dataset and instantly perform **Exploratory Data Analysis (EDA)** with interactive visualizations, summary statistics, missing value analysis, and relationship exploration.

---

## 🚀 Project Overview

The **Dataset Intelligence Dashboard** is designed to simplify data exploration for data analysts, students, and business professionals. Simply upload a dataset, and the dashboard automatically generates meaningful insights through interactive charts and analytics.

This project helps users understand their data before moving to Machine Learning, Business Intelligence, or Statistical Analysis.

---

## ✨ Features

### 📁 Dataset Upload
- Upload any CSV dataset
- Automatic dataset loading
- Supports large datasets

### 📈 Dataset Overview
- Total Rows
- Total Columns
- Numeric Features
- Categorical Features
- Missing Data Percentage

### 🔍 Data Preview

- Scrollable dataset preview

### 📊 Summary Statistics
Automatically calculates:

- Minimum

### ⚠ Missing Value Analysis
- Missing values by column

### 🔗 Relationship Analysis
- Correlation Matrix
- Scatter Plots
- Heatmaps

### 📂 Category Analysis
- Category Frequency

- Value Distribution

- Time-based visualizations
- Numerical trend exploration

### 🎯 Filters
- Filter by categorical columns
- Filter by numeric ranges

# ⚙ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/Dataset-Intelligence-Dashboard.git
```

Move into the project directory

```bash
cd Dataset-Intelligence-Dashboard
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
